# CISC101-Group--PaperSummarizer
Overview:
Paper Summarizer is a tool that takes an academic paper or long document and creates a clear, section-based summary. It was developed for an Intro to Computer Science project and is designed to be used through Microsoft Copilot.

Features:
Summarizes long papers into key sections
Highlights main ideas
Easy to use within Microsoft Copilot

How to Use:
Open Microsoft Copilot
Paste the paper or text
Run the summarizer prompt
Read the generated section-by-section summary
